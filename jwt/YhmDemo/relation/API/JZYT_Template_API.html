Jzyt Template API(示例) 1.3

获取用户列表

  • 请求URI:
    /user/get_user_list GET REST

    Content-Type: application/json

    请求Body参数:
    参数名 参数值 Required
    pageNumber {pageNumber} Yes
    请求Body示例:

    {"pageNumber":"1"}


    响应:

    {"success":true,"errors":[]} (Get user successful)

    {"success":false,"errors":["获取列表失败!!!"]} (Get user has errors)

添加用户

  • 请求URI:
    /user/do_add_user POST REST

    Content-Type: application/json

    请求Body参数:
    参数名 参数值 Required 备注
    username {username} Yes Member login name, length >= 6
    password {password} Yes Member login password, length >= 8
    rePassword {rePassword} Yes Member login rePassword, length >= 8
    roleName {roleName} Yes Member role name
    请求Body示例:

    {"username":"user01","password":"12345678","rePassword":"12345678","roleName":"ADMIN, USER"}


    响应:

    {"success":true,"errors":[]} (Save user successful)

    {"success":false,"errors":["添加失败！！！"]} (Save user has errors)

编辑用户

  • 请求URI:
    /user/do_update_user POST REST

    Content-Type: application/json

    请求Body参数: tr>
    参数名 参数值 Required 备注
    uuid {uuid} Yes Member uuid not null
    username {username} Yes Member username, length >= 6
    roleName {roleName} Yes Member role name
    请求Body示例:

    {"uuid":"9db94effebca22b2f096d3daf4715a1edD36WIBaDcV","username":"user01","roleName":"ADMIN,USER"}


    响应:

    {"success":true,"errors":[]} (Update user successful)

    {"success":false,"errors":["用户修改失败！！！"]} (Update user has errors)
